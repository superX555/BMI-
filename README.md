BMI 健康追蹤系統 (BMI Health Tracker) v2.0
這是一個基於 C# Windows Forms (WinForms) 開發的輕量化個人健康管理工具。使用者可以輸入個人基本資料（姓名、身高、體重），系統會自動計算 BMI (身體質量指數)，並提供即時的診斷建議與歷史紀錄功能。

🌟 主要功能
即時 BMI 計算：根據世界衛生組織標準，自動換算並顯示 BMI 數值。

視覺化診斷標示：

🔵 藍色：體重過輕 (BMI < 18.5)

🟢 綠色：正常範圍 (18.5 ≤ BMI < 24)

🟠 橘色：過重 (24 ≤ BMI < 27)

🔴 紅色：肥胖 (BMI ≥ 27)

介面紀錄清單：使用 DataGridView 元件，即時在介面上呈現本次執行的所有計算歷史（含姓名、身高、體重、時間）。

永久存檔功能：自動將每筆紀錄同步寫入本地端 BMI_Records.txt 檔案，確保資料不因程式關閉而遺失。

歷史快速檢視：內建一鍵開啟紀錄檔功能，方便透過記事本進行後續編輯或導出。

🛠️ 核心技術
開發語言：C# (.NET Framework / .NET Core WinForms)

介面設計：Windows Forms Designer

資料儲存：System.IO (File Stream / AppendAllLines)

事件驅動：Delegate & Event Handling (按鈕點擊、表單載入)

錯誤處理：Try-Catch 異常擷取（防止非法數字輸入導致程式崩潰）

🚀 如何使用
輸入資料：在「個人資料輸入」區塊輸入使用者的姓名、身高(cm)及體重(kg)。

執行計算：點擊「計算並儲存」按鈕。

查看結果：

上方圓框會顯示當前 BMI 數值與狀態。

下方表格會新增一筆完整的紀錄資料。

異常數據（過輕/過重/肥胖）在表格中會以 紅色粗體 醒目標示。

查閱歷程：點擊「查看歷史紀錄」可開啟包含所有歷史數據的 BMI_Records.txt。
